Lab & CTF Writeups

TryHackMe: Blue

Problem: Exploit the "EternalBlue" vulnerability (MS17-010) on a Windows 7 machine to gain system-level access.

Approach: Used Nmap to discover open SMB ports. Searched Metasploit for the MS17-010 exploit module, configured the payload, and successfully executed it to get a reverse shell with SYSTEM privileges.

Tools Used: Nmap, Metasploit Framework.

Lessons Learned: A powerful reminder of the critical importance of timely patching for known, high-severity vulnerabilities.

Forensics: Disk Image Analysis

Problem: Analyze a provided disk image to find a hidden flag within a deleted file.

Approach: Mounted the disk image using FTK Imager and ran Autopsy to analyze the file system. Searched through unallocated space and located fragments of a deleted PNG image. Carved the file and found the flag embedded in the image's metadata.

Tools Used: Autopsy, FTK Imager, ExifTool.

Lessons Learned: Gained hands-on experience with file carving and metadata analysis, crucial skills in digital forensics investigations.
