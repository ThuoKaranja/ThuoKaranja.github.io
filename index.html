CTF: Basic Pentesting

Problem: Gain root access to a vulnerable Linux machine by exploiting a web application vulnerability.

Approach: Started with Nmap scan to identify open ports (80, 22). Found a vulnerable web application, used SQL injection to bypass login, uploaded a reverse shell, and escalated privileges using a kernel exploit.

Tools Used: Nmap, Burp Suite, Metasploit, LinEnum.

Forensics: Stolen Data

Problem: Analyze a disk image to determine what data was exfiltrated by a disgruntled employee.

Approach: Used Autopsy to analyze the disk image. Examined browser history, deleted files, and registry keys. Discovered a USB drive was used and found fragments of the stolen documents in unallocated space.

Tools Used: Autopsy, FTK Imager.

Lessons Learned: Deepened my understanding of digital forensics workflows and how data can be recovered even after deletion.
